Kebijakan Privasi

Terakhir diperbarui: 3 Januari 2026

1. Pendahuluan

Aplikasi Lolos CPNS - Simulasi CAT SKD ("Aplikasi") dikembangkan untuk membantu pengguna mempersiapkan ujian CPNS. Kebijakan privasi ini menjelaskan bagaimana kami menangani data pengguna.

2. Data yang Dikumpulkan

Aplikasi ini tidak mengumpulkan data pribadi dari pengguna. Semua data latihan dan progress disimpan secara lokal di perangkat pengguna.

  • Tidak ada pendaftaran akun yang diperlukan
  • Tidak ada data yang dikirim ke server eksternal
  • Tidak ada pelacakan lokasi
  • Tidak ada iklan

3. Penyimpanan Data Lokal

Aplikasi menyimpan data berikut secara lokal di perangkat Anda:

  • Riwayat latihan dan skor
  • Pengaturan aplikasi
  • Progress belajar

Data ini tidak pernah dibagikan dengan pihak ketiga dan dapat dihapus kapan saja dengan menghapus aplikasi.

4. Izin Aplikasi

Aplikasi ini memerlukan izin kamera hanya untuk fitur deteksi wajah saat simulasi ujian. Data kamera diproses secara lokal dan tidak disimpan atau dikirim ke mana pun.

5. Keamanan

Karena tidak ada data yang dikumpulkan atau ditransmisikan, risiko keamanan data minimal. Semua pemrosesan dilakukan secara lokal di perangkat Anda.

6. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan akan dipublikasikan di halaman ini.
